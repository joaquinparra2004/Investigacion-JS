2) ¿Cómo se integra JavaScript a HTML?.

Para poder lograr integrar correctamente JavaScript a HTML lo tenemos que vincular desde el HTML hacia el JS a través de 3 etiquetas, las cuales dos de estas pueden ser declaradas en el header y una que puede ser declarada dentro del body del HTML.
  • Poder enlazar con el atributo "src", el cual se coloca en el header con la etiqueta de script, donde permite crear una ruta hacia un archivo externo o recurso que quieres enlazar a tu documento HTML.
  • Poder enlazar directamente desde el HTML con la etiqueta script, donde pondremos todo el código de JS que tengamos dentro de esta etiqueta la cual estará ubicada en el body del HTML.
  • Poder enlazar directamente desde el HTML con la etiqueta script. esta forma a diferencia de la anterior, se tiene que declarar dentro del head usando la misma metodología donde utilizamos la etiqueta script y ponemos el código JS dentro de ella.

3) ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.

La forma de manejar las variables dentro de JS son tres, donde estas mismas se declaran a partir ya sea:
Var, donde se declara la variable opcionalmente la inicia a un valor. Esta sintaxis se puede utilizar para declarar variables locales y globales, dependiendo del contexto de ejecución. Por ejemplo, var x = 12;
Let, donde se declara una variable local con ámbito de bloque, opcionalmente la inicia a un valor. Esta sintaxis se puede utilizar para declarar una variable local con ámbito de bloque. Por ejemplo, let y = 13;
Const, donde se declara un nombre de constante de solo lectura y ámbito de bloque. También comparte la misma sintaxis que let.

Para manejar los tipos de datos se utiliza el ultimo estándar de ECMAScript que define siete tipos de datos:

  • Booleano - True y False
  • null - Palabra clave especial que denota un valor nulo.
  • undefined - Propiedad de alto nivel que su valor no está definido.
  • Number - Un numero entero o con una coma flotante.
  • Biglnt - Un numero entero con precisión arbitraria.
  • String - Secuencias de caracteres que hacen el valor de texto.
  • Symbol - Un tipo de dato cuyas instancias son únicas e imutables.
Para manejar correctamente las Expresiones regulares, donde tiene la posibilidad de poder usar grupos nombrados usando una expresión regular. Un match de esta misma expresión va a retornar un objeto. Si alguna parte de la expresión está dentro de los paréntesis, es agarrado por el grupo y almacenado en el grupo que haya hecho match.Por ejemplo: const data = /([0-]{9})-([0-9]{2})-([0-9]{2})/; const matchedObject = eventDate.exec('2019-04-03');.

7) ¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?.

El objeto DOM (Document Object Model) en JavaScript, que es la misma estructura del documento HTML. Una página HTML la cual está formada por múltiples etiquetas, anidadas una dentro de otra, formando un árbol de etiquetas relacionadas entre sí.
Esta estructura, se puede modificar de forma dinámica desde JavaScript. Añadiendo nuevas etiquetas, modificando o eliminando otras, cambiando sus atributos HTML. El DOM visualiza el documento de HTML como un árbol de tres nodos, donde cada nodo representa un documento de HTML.

Para utilizar el DOM en JS, hay diferentes métodos para seleccionar un elemento en el HTML.

  • getElementById () Toma una etiqueta que tenga id en HTML.
  • querySelector () Toma uno o más selectores CSS.
  • querySelectorAll () Encuentra todos los elementos que coinciden con el selector de CSS y devuelve una lista de todos esos nodos.
Para agregar nuevos elementos al documento se utiliza document.createElement () donde se agregan nuevos elementos al árbol del DOM.Lo mismo pasa si queremos eliminar un elemento donde usaremos ().

9) ¿Cuáles son todos los eventos que trabaja JS?.

Los eventos dentro de JS no son más que una notificación de que alguna característica interesante acaba de ocurrir, las cuales mayormente se relacionan con el usuario. Algunos eventos en los cuales trabaja JS son estos:

  • onblur – Un elemento pierde el foco
  • onclick – Pulsar y soltar ratón
  • onmouseout – Un elemento ha sido modificado.
  • onmouseover- Un elemento obtiene el foco.
  • onkeydown – Pulsar una tecla y no soltarla.
  • onkeypress – Pulsar una tecla.
  • onkeyup – Soltar una tecla pulsada.
  • onreset – Inicializar el formulario.
  • onsubmit – Enviar el formulario.
  • onresize – Modificar el tamaño de la ventana.
